# felicios-tech-site
Repositório da landing page oficial da Felicio's Tech, laboratório especializado em manutenção física de dispositivos eletrônicos localizado na Taquara, Rio de Janeiro.  

🛠️ Especialidades do Laboratório
Hardware de Celulares: Troca de telas, baterias e reparo em conectores.  

Notebooks e PCs: Manutenção interna, reparo de dobradiças e upgrades físicos.  

Consoles de Vídeo Game: Limpeza técnica, troca de pasta térmica e reparos de hardware.  

🏛️ Transparência e Conformidade
Este projeto foi desenvolvido seguindo as diretrizes de transparência do Google Ads para assistências multimarcas:  

Garantia: Todos os serviços possuem garantia legal de 90 dias conforme o Código de Defesa do Consumidor.  

Independência: Somos uma assistência técnica independente e não possuímos vínculo direto (autorizada) com fabricantes como Apple, Samsung ou Dell.  

Localização: Atendimento presencial na Estrada do Outeiro Santo, 405 - Taquara, RJ.  

🚀 Tecnologias
HTML5 / CSS3  

Font Awesome para ícones técnicos  

Hospedagem: GitHub Pages com SSL ativo (obrigatório para Google Ads)
